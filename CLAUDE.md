# Project: Leonistic

A landing page for Leonistic — building software, analyzing data, and explaining technology, one clear solution at a time. The design should feel confident-direct: clean, modern, and trustworthy.

## Always

- Use the frontend-design skill before writing any front-end code.
- Reference `brand_assets/` for the logo, colors, and brand info. The full brand identity (colors, typography, UI elements, voice) is in `brand_assets/brandidentity.png`; the tagline and audience are in `brand_assets/brand.md`.

## Stack

- HTML, CSS, vanilla JavaScript. No framework on Day 2.
- Deploy on Vercel via GitHub auto-deploy.

## Never

- Use placeholder Lorem Ipsum text.
- Ship designs that look "AI vibe coded": generic gradient stock backgrounds, fake stats, badge-and-pill templates with no content.
- Modify files in `brand_assets/`.
